## 7. 安全架构

### 7.1 安全机制

| 层面 | 机制 | 实现 |
|------|------|------|
| 认证 | Cookie-based Session | FastAPI Session |
| 授权 | 路由守卫 | Vue Router Guards |
| 传输 | HTTPS | Nginx SSL |
| 输入 | 参数校验 | Pydantic Models |
| SQL | 参数化查询 | SQLAlchemy ORM |
| XSS | 转义输出 | Vue Template |
| CSRF | SameSite Cookie | FastAPI CORS |
